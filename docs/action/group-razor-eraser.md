# 【合集】Razor eraser - 橡皮擦模式（按住键 + 右键拖动删区域）

一个临时"橡皮擦"模式——按住触发它的快捷键期间，**右键拖动就能直接删掉划过的区域**。底层借用 REAPER 的 Razor Edit（剃刀编辑）来圈选并删除。共有两个变体，区别只在**删除区域是否吸附（snap）**。

---

## 两个变体对照

| Action List 里的精确显示名 | 右键拖动选区是否吸附 |
|---|---|
| **Razor eraser (hold key + right-drag to delete areas)** | **会吸附**到网格 / Item 边缘等吸附点 |
| **Razor eraser ignoring snap (hold key + right-drag to delete areas)** | **不吸附**，落点完全跟手 |

---

## 怎么用

1. 给这个 Action 绑一个快捷键
2. **按住**那个快捷键不放
3. 用**右键在 Arrange 里拖动**，框出要删的区域
4. **松开右键** → 框住的那段立刻被删除
5. 想擦下一处就再右键拖一次
6. **松开快捷键** → 退出橡皮擦模式

可以理解成"按住键 = 拿起橡皮擦，松开键 = 放下橡皮擦"。

---

## 共有行为细节

- 进入橡皮擦模式时会**先清掉你已有的 Razor 选区**，避免一进来就误删
- 删除是 REAPER 标准的"Delete razor area"行为：删 Item 内容、清空所选区域
- 每**松开一次右键**就是**一次 Undo**——擦错了按一次 Ctrl+Z 还原那一笔
- 退出模式时会**清掉残留的 Razor 选区**，并把右键修饰键设置**恢复成你原来的样子**（橡皮擦期间临时借用了右键，退出后归还）
- 模式生效期间，**右键被临时占用**来画删除区域，原本的右键菜单暂时不出
- 鼠标旁会跟一行小提示字，告诉你当前在橡皮擦模式

---

## 两个变体的唯一区别

唯一区别是右键拖动出的选区**是否吸附**：

- **Razor eraser**（吸附版）：选区**会吸附**到网格 / Item 边缘等吸附点
- **Razor eraser ignoring snap**（忽略吸附版）：选区**不吸附**，落点完全跟手

其余行为（进入时清旧选区、每松一次右键 = 一次 Undo、退出时清残留并归还右键设置、鼠标旁提示字）两者**完全相同**。

### 什么时候用忽略吸附版

- 要删的区域**不在网格 / Item 边界上**，吸附老把你拽偏的时候
- 想徒手精确刮掉一小段，不被吸附干扰
